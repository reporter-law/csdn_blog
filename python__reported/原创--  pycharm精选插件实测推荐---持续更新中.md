# pycharm精选插件实测推荐---持续更新中

pycharm插件能够实现一些必要的功能，但是也会减缓pycharm启动及运行速度。<br/> 以下插件为通用的必不可少的插件

一、安装方式：<br/> （原版pycharm）file&gt;setting&gt;Plugins<br/> （汉化版pycharm）文件&gt;设置&gt;Plugins
